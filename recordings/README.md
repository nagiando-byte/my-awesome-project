# 録画ファイルをここに置いてください

このフォルダは、Zoom録画ファイルを保存するためのフォルダです。

## 使い方

### 方法1: 手動で録画をコピー

Zoomの録画フォルダから、このフォルダにコピー:

```bash
# macOSの場合（デフォルトのZoom保存先）
cp ~/Documents/Zoom/[日付]\ [会議名]/zoom_0.mp4 ./

# または、Finderでドラッグ&ドロップ
```

### 方法2: 監視モードを使う

このフォルダを監視して、新しいファイルが追加されたら自動で文字起こし:

```bash
cd ..
npm run transcribe -- -w recordings
```

### 方法3: Zoomの保存先を直接監視

```bash
npm run transcribe -- -w ~/Documents/Zoom
```

## 対応ファイル形式

- `.mp4` (推奨)
- `.mov`
- `.avi`
- `.mkv`
- `.webm`

## テスト用

短い録画（1-2分）でテストすることをお勧めします:
1. Zoomで短いテスト録画を作成
2. このフォルダにコピー
3. 文字起こしを実行

```bash
npm run transcribe -- -i recordings/test.mp4
```

## ヒント

- ファイル名に日本語を使用しても大丈夫です
- ファイルサイズが大きい場合（25MB以上の音声）、処理に時間がかかる場合があります
- 長時間の録画（2時間以上）は分割を検討してください
